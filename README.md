# smartcord

A chatbot, basically a wrapper for https://api.affiliateplus.xyz.

### Basic Usage:

```
const smartcord = require("smartcord")

smartcord.chat("Hey", {
    botName: "some name",
    ownerName: "some name",
    userName: "some name"
})
.then(m => console.log(m)) // Hey, my friend! Do you want me to tell you a joke?
```
